Uso de Entidades de Caracteres HTML

Los caracteres menores que y mayores que se usan para las etiquetas en HTML, así que si queremos mostrar esos caracteres, necesitamos usar sus entidades: < (menor que) y > (mayor que).

Para mostrar comillas dobles dentro de un texto, usamos ". Por ejemplo: "Hola, mundo!"
Para comillas simples, usamos '. Ejemplo: I'm learning HTML.

El símbolo de ampersand (&) se representa con &. Ejemplo: AT&T.

Para caracteres especiales en otros idiomas, como la ñ en español: español (español) o la letra alemana ß: Straße (Straße).

Símbolos de moneda: el dólar ($), el euro (€), y la libra esterlina (£).

Símbolos matemáticos: ± (más/menos), × (multiplicación), ÷ (división), y √ (raíz cuadrada).

Ejemplos de entidades numéricas: © (© para copyright), ® (® para marca registrada), y ♥ (♥ para un corazón).

Algunas flechas: ← (←), ↑ (↑), → (→), ↓ (↓).
Otros símbolos: ♥ (♥), ♣ (♣), ♦ (♦), ♠ (♠).
